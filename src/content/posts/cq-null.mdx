# 空

> # 空(śūnya)是什么。

何不尝让此文为空，亦也不为一种空。

新开这篇文章时，想的是仅仅以此来记录脑中的空。回想过往岁月，脑子空空似乎是最常见的了，这是大脑的一种保护机制吗？还是其他的什么呢？也罢，向来语文不好，恐无法描述。

我此刻提笔欲下笔却无从下笔，也为空。

论具象与抽象来说，我想抽象大抵是要留空吧。正如水墨画中的留白一样，以无相表达意象，言有尽而意无穷。

奈何才疏浅薄，无法知其意。近日观佛，便以`色即是空，空即是色`来化作句号。
